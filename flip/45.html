🎨 GPU = Graphics Processing Unit 🖼️ Pelukis Super Cepat!

💎 Inilah GPU – Si Pelukis Keren!

🖼️ Render Gambar 🎮 Jalankan Game 📺 Putar Video 🌐 Grafis 3D 🤖 AI & ML

🎮 Apa yang GPU Lakukan?

🎨

Pelukis Super Cepat

GPU melukis jutaan titik (pixel) di layarmu setiap detik. Setiap frame game = satu lukisan baru!

👆 Tap untuk tahu lebih!
⚡

FPS – Frame Per Detik

GPU yang kuat bisa menghasilkan 60-240 gambar per detik! Makin banyak, makin halus gerakannya.

👆 Tap untuk tahu lebih!
🧠 vs 🎨

CPU vs GPU

CPU itu otak — pintar tapi kerjain satu-satu. GPU itu pelukis — bisa kerjain ribuan hal sekaligus!

👆 Tap untuk tahu lebih!
💾

VRAM – Ingatan Pelukis

GPU punya memori sendiri bernama VRAM! Ini tempat menyimpan tekstur, warna, dan gambar yang sedang dilukis.

👆 Tap untuk tahu lebih!

📊 GPU Lemah vs GPU Kuat

😟 GPU Lemah
❌ Gambar buram & kotak-kotak
❌ Game patah-patah (lag!)
❌ Hanya 10-30 FPS
🔥 GPU Kuat
✅ Gambar tajam & indah
✅ Game super halus & lancar
✅ 60-240+ FPS! 🚀
