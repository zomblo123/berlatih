🧠 3 Fakta Penting RAM

⚡

Super Cepat

RAM bisa kirim data ke CPU jutaan kali lebih cepat dari Hard Disk! Seperti lempar bola ke tangan vs kirim surat. 📮

💨

Sementara

Semua isi RAM langsung hilang saat komputer dimatikan. Itu kenapa kamu harus "Save" dulu sebelum matiin komputer! 💾

🤝

Tim CPU

RAM adalah asisten paling dekat CPU. Tanpa RAM, CPU tidak bisa kerja! Mereka tim impian. 🧠💾

💾

🤯 Fakta Gila Hari Ini!

RAM pertama di dunia (tahun 1947!) seukuran lemari besar dan cuma bisa simpan 4.096 bit data. RAM HP-mu sekarang 4 GB = 8 miliar kali lebih besar... dan seukuran kuku jarimu! 😲🔬
