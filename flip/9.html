// Halaman 02
Setiap Hari Kita Membuat Keputusan! 🤔
Setiap hari, tanpa sadar kita selalu membuat pilihan:
🌧️
Kalau...
Hujan → Pakai payung
😋
Kalau...
Lapar → Makan dulu
Komputer pun bisa melakukan hal yang sama! Inilah yang disebut If-Else — artinya "Kalau Begini, Maka Begitu!"
// Halaman 03
Robot Penjaga Pintu yang Cerdas! 🚪
Bayangkan kamu punya robot penjaga pintu. Kamu kasih perintah:
KALAU tombol_hijau ditekan:
    → buka_pintu()
KALAU TIDAK:
    → tetap_tutup()
Robot akan memeriksa kondisi dulu, baru memutuskan apa yang harus dilakukan. Itulah cara kerja If-Else!
// Halaman 04
Flowchart: Peta Keputusan Komputer! 🗺️
Di coding, keputusan bekerja seperti peta jalan bercabang:
# Struktur If-Else dasar
if (kondisi_benar):
    lakukan ini
else:
    lakukan itu
Komputer memeriksa kondisi → kalau BENAR, ikuti jalur kiri → kalau SALAH, ikuti jalur kanan. Sesederhana itu!
// Halaman 05
If-Else di Game! Seru Banget! 🎮
Tahukah Aslan? Di dalam game favoritmu, ada ribuan If-Else yang bekerja setiap detiknya!
if tombol_atas ditekan:
    → karakter_lompat()
if tombol_kanan ditekan:
    → gerak_kanan()
if kena_musuh:
    → game_over()
Tanpa If-Else, karakter game hanya diam saja — tidak bisa bergerak ke mana-mana! If-Else adalah nyawanya game! 🎯
// Halaman 06
Robot Quiz! Jawab Benar atau Tidak? 🎯
Aslan bisa membuat robot kuis yang cerdas dengan If-Else:
if jawaban == "Jakarta":
    print("🎉 Kamu Hebat! +10 poin!")
else:
    print("💪 Coba Lagi, Pasti Bisa!")
Robot akan otomatis memberi respons yang tepat! Ini bukan sihir — ini If-Else! 🤖
🧠
Setiap aplikasi belajar seperti Duolingo, Khan Academy, dan game edukasi — semua memakai If-Else untuk memberikan respons yang berbeda tergantung jawabanmu!
// Halaman 07
Di Scratch: Balok If-Else yang Mudah! 🟡
Di Scratch, If-Else terlihat seperti puzzle warna-warni yang bisa disusun:
🔶 if <hujan = BENAR?> then
☂️ pakai payung
🔴 else
⛅ main di luar
Tinggal taruh balok "if", isi kondisinya, lalu tambah balok aksinya. Komputer akan otomatis memilih jalur yang tepat! Semudah bermain Lego! 🧩
// Halaman 08
Robot Pembantu Rumah yang Pintar! 🏠
Bayangkan robot pembantu yang bisa membantu Bunda di rumah:
if piring_kotor:
    → cuci_piring()
elif lantai_kotor:
    → sapu_lantai()
else:
    → istirahat() 😴
Robot pakai keputusan bersyarat agar tidak boros energi — hanya bekerja kalau benar-benar diperlukan! Cerdas, kan? 🤖💡
// Halaman 09
Keputusan Bersyarat: Dua Kondisi! 🔗
Kadang kita butuh dua kondisi sekaligus. Ini namanya AND (dan)!
if nilai_mtk_bagus AND nilai_bi_bagus:
    print("🎮 Boleh main game 1 jam!")
elif nilai_mtk_bagus:
    print("📖 Belajar B.Inggris dulu")
else:
    print("📐 Belajar Matematika dulu")
Ini inti dari pemrograman yang pintar — keputusan yang mempertimbangkan banyak hal sekaligus! 🧠
// Halaman 10
If-Else di Game: Strategi Tanpa Henti! ⚔️
Game RPG yang kamu mainkan penuh dengan keputusan If-Else yang berjalan setiap detik:
if musuh_dekat AND hp > 50:
    → serang_musuh() ⚔️
elif hp < 20:
    → lari_sembunyi() 🏃
else:
    → pakai_obat() 💊
Karakter yang "pintar" dalam game — itu hasil kerja ribuan If-Else yang dirancang programmer! Sekarang Aslan juga bisa bikin itu! 🎮🚀
// Halaman 11
Aslan Bisa Mengontrol Komputer! 🎛️
Dengan memahami If-Else, kamu bisa memerintah komputer untuk melakukan hal yang tepat di waktu yang tepat!
  • 🔘 Tombol hijau ditekan? → Robot maju ke depan
  • 🔴 Tombol merah ditekan? → Robot berhenti segera
  • 🟡 Baterai lemah? → Robot cari tempat charging
  • ✅ Tugas selesai? → Robot lapor dan istirahat
Mulai dari yang sederhana, nanti kamu bisa bikin program yang sangat canggih! 🚀 Semua programmer hebat juga mulai dari sini!
// Halaman 12
Aslan: Kapten Keputusan! ✈️
Sistem autopilot pesawat menggunakan If-Else yang sangat canggih:
if cuaca_buruk:
    → ganti_rute() 🌩️
elif bahan_bakar < 20%:
    → mendarat_darurat() ⚠️
else:
    → lanjutkan_penerbangan() ✈️
If-Else adalah salah satu kekuatan terbesar coding. Dengan ini, kamu bisa membuat sistem yang berpikir dan bereaksi seperti manusia — tapi jauh lebih cepat dan tepat! 🧠⚡
