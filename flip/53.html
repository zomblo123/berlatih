🧠
👩‍🍳

CPU = Koki Pintar

Koki yang memasak dan mengolah semua bahan menjadi makanan lezat. Tanpa koki, bahan-bahan hanya diam saja!

📋
🍽️

RAM = Meja Makan

Meja tempat koki menaruh bahan-bahan yang sedang dipakai. Kalau mejanya kecil, tidak bisa masak banyak sekaligus!

🗄️
🧺

Storage = Kulkas Besar

Kulkas besar tempat menyimpan semua bahan makanan. Walaupun listrik mati, bahan di kulkas tetap ada!

😄

✨ Rangkuman Hari Ini!

🧠

CPU = Otak komputer yang memproses semua perintah. Cepat sekali tapi tidak menyimpan data.

📋

RAM = Meja kerja sementara. Menyimpan data yang sedang dipakai CPU. Kalau komputer mati, datanya hilang.

🗄️

Storage = Lemari besar untuk menyimpan data selamanya. Foto, video, game — semua di sini!

🤝

Kerja sama: Storage kirim data → RAM tampung → CPU proses → Hasil tampil di layar! 🎉
